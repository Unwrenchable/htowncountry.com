# Hometown Country Kitchen Website

Welcome to the Hometown Country Kitchen restaurant website! This site showcases Henderson, Nevada's favorite spot for homestyle comfort food.

## About the Restaurant

**Hometown Country Kitchen** (H-Town Country) is a locally beloved comfort food restaurant serving Henderson, Nevada with:

- **Location:** 894 S Boulder Hwy, Henderson, NV 89015
- **Phone:** (702) 567-0612
- **Hours:** 7:00 AM - 7:00 PM Daily

## Features

- Breakfast served all day
- Homestyle American comfort food
- Family-friendly atmosphere
- Wheelchair accessible
- Curbside pickup and takeout available

## Website Features

This website includes:

- **Responsive Design:** Works on desktop, tablet, and mobile devices
- **Menu Showcase:** Complete menu with breakfast, lunch, dinner, and desserts
- **About Section:** Information about the restaurant's hometown values
- **Contact Information:** Easy access to location, phone, and hours
- **Smooth Navigation:** Easy-to-use navigation with smooth scrolling

## Technology

- Pure HTML5, CSS3, and JavaScript
- No external dependencies (except Google Fonts)
- Responsive and accessible
- Fast loading and optimized

## Color Scheme

The website uses a warm, country-inspired color palette:
- Saddle Brown (#8B4513)
- Chocolate (#D2691E)
- Peru (#CD853F)
- Cornsilk (#FFF8DC)
- Goldenrod (#DAA520)

## How to View

Simply open `index.html` in any modern web browser to view the website.
