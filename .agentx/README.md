# AgentX Pack for htowncountry.com

This repository is upgraded with AgentX capabilities.

## Included
- `agents.json`: merged core + imported + custom per-repo agents
- `access_profiles.json`: safe/balanced/power profiles
- `agency_import.json`: imported agents from agency-agents

## Suggested commands
```bash
agentx find htowncountry.com
agentx check htowncountry-com-implementation-pilot --profile balanced
agentx check htowncountry-com-orchestrator --profile power
```
